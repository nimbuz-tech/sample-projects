# Larger Sample Next.js 14 App

This is a slightly larger example Next.js application using the App Router.

## Features

- App Router (`app` directory)
- Shared root layout with header & footer
- Static pages: Home, About, Contact
- Blog listing page and dynamic blog post route (`/blog/[slug]`)
- Nested dashboard layout with `/dashboard` and `/dashboard/settings`
- Simple API route at `/api/hello`

## Getting Started

```bash
npm install
npm run dev
```

Then open http://localhost:3000 in your browser.
