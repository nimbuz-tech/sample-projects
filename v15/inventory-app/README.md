# Inventory App (Angular)

Modules/Features included (no login/auth):
- Dashboard
- Products (CRUD)
- Stock (ledger + IN/OUT/ADJUST)
- Orders (draft + confirm reduces stock)
- Reports (category summary + top stock value)

## Run locally

```bash
npm install
npm start
```

Then open: http://localhost:4200

## Notes
- Uses browser LocalStorage for persistence (demo-friendly).
- Responsive UI (sidebar collapses on mobile).
